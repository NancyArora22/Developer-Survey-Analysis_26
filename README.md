# Developer Survey Analysis – Riipen Project(IBM)

## Objective
Analyze developer survey data to identify trends in job satisfaction and technology usage.

## Tools Used
Python, SQL, IBM Cognos

## Data Cleaning
- handled missing values
- removed duplicates
- validated data consistency
- standardized formats

## Analysis
- compared current vs future technology usage
- analyzed trends in languages, platforms, databases

## Key Insights
- identified skill gaps between current and future demand
- highlighted trends in developer preferences

## Learning
- importance of data validation before analysis
- applying analytical techniques to real-world datasets
